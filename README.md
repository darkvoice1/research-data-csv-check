# 科研数据CSV合规检测工具
一款轻量级的科研数据CSV文件合规检测工具，支持10条全领域通用检测规则，一键上传、自动检测、结果可视化展示。

## 功能特点
- 📊 支持10条通用合规检测规则（必填列、日期格式、数值合法性、重复值等）；
- 🚀 前后端分离，操作简单，非技术人员也能使用；
- 🔧 高度可配置，只需修改配置文件即可适配不同数据场景；
- 📝 检测结果详细，精准定位违规行和问题类型。

## 检测规则
1. 必填列检测：检查样本编号等核心列是否存在且非空；
2. 日期格式检测：校验日期列是否为YYYY-MM-DD/YYYYMMDD格式；
3. 数值列检测：检查数值列是否为数字且非负数；
4. 重复值检测：检查唯一标识列是否存在重复值；
5. 列名规范检测：禁止列名包含特殊字符、全角空格；
6. 空行检测：检查CSV中是否存在纯空行；
7. 枚举值检测：检查固定取值列（如样本类型）是否符合预设范围；
8. 数据长度检测：检查字符长度不超过预设最大值；
9. 特殊字符检测：禁止文本列包含敏感特殊字符（\ / : * ? " < > |）；
10. 全角字符检测：检查数字/字母是否为半角格式。
